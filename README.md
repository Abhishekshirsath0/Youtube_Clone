# Youtube_Clone
A simple YouTube clone built with HTML, CSS, and Bootstrap.
## Features

- Responsive design
- Video player
- Navigation bar
- Search bar
- Video grid

## Technologies Used

- HTML
- CSS
- Bootstrap

## Setup

1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/youtube-clone.git
    ```

2. Open `index.html` in your browser to view the project.
